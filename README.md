# 📁 Thư mục IMAGES - Hướng dẫn đặt ảnh

Để trang web hiển thị đúng, bạn cần đặt ảnh vào đây với tên file đúng như sau:

## Ảnh cần chuẩn bị (5 ảnh):

| Tên file        | Dùng cho              | Kích thước khuyến nghị |
|-----------------|----------------------|------------------------|
| cinema-logo.jpg | Logo nhỏ góc header  | 80x80 px               |
| phim1.jpg       | Ảnh bộ phim thứ 1    | 400x220 px             |
| phim2.jpg       | Ảnh bộ phim thứ 2    | 400x220 px             |
| phim3.jpg       | Ảnh bộ phim thứ 3    | 400x220 px             |
| phim4.jpg       | Ảnh bộ phim thứ 4    | 400x220 px             |
| rap-phim.jpg    | Ảnh rạp chiếu phim   | Bất kỳ kích thước nào  |

## Cách đặt ảnh:
1. Sao chép ảnh của bạn vào thư mục này (images/)
2. Đổi tên file ảnh theo đúng tên ở bảng trên
3. Mở lại trình duyệt để xem kết quả

## Lưu ý:
- File ảnh có thể là .jpg, .jpeg, .png, .webp
- Nếu ảnh tên khác, bạn có thể chỉnh trong file HTML
  Ví dụ: <img src="images/TEN-ANH-CUA-BAN.jpg" ...>
